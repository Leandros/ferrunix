# Installation

Install [`ferrunix`] by adding the following to your `Cargo.toml`.

```toml
[dependencies]
ferrunix = "0.3"
```

or by running `cargo add`:

```bash
cargo add ferrunix
```

Once installed, the next step is understanding the [core concepts].

[`ferrunix`]: https://github.com/Leandros/ferrunix
